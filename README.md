# P&L Habi — Dashboard (protegido)

Este repositorio publica **únicamente** una versión cifrada del dashboard de P&L,
servida vía GitHub Pages. El contenido está cifrado con AES-256-GCM y requiere
contraseña para visualizarse; sin ella, `index.html` no revela ningún dato.

El pipeline de datos, las consultas y la lógica de negocio viven en un repositorio
privado aparte. Aquí no hay datos en claro ni código interno.

**Ver el dashboard:** abre la URL de GitHub Pages de este repo e ingresa la contraseña.
